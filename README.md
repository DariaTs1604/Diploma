# Инструкция по начальной настройке
## Подсистема «Обслуживание клиентов»
1.	Создать бота и группу в Телеграмм, для отправки уведомлений специалистам о создании/изменении документа «Обслуживание клиентов».
2.	Заполнить константы:
•	Идентификатор группы для оповещения - идентификатор созданной группы в Телеграмм;
•	Токен управления телеграм-ботом – токен, назначенный при создании телеграмм-бота;
•	Номенклатура абонентская плата – необходимо выбрать подходящее значение из справочника «Номенклатура» или создать новое;
•	Номенклатура работы специалиста - необходимо выбрать подходящее значение из справочника «Номенклатура» или создать новое;
## Подсистема «Зарплата и кадры»
3.	Заполнить регистр сведений «Условия оплат сотрудников». Необходимо указать значения окладов сотрудников и процент премии от выполненных работ специалистов. 
4.	Создать в справочнике «Графики работ» необходимый график работы сотрудников (пятидневка, шестидневка и т.п.). С помощью обработки «Заполнение графика», расположенной в разделе «Сервис» подсистемы «Зарплата и кадры», необходимо заполнить регистр сведений «Графики работ». 
5.	Создать элементы справочника «Сотрудники» для каждого сотрудника, если это не было сделано ранее. 
6.	Назначение Пользователям соответствующий профиль группы доступа:
- Менеджер – работа с документом «Обслуживание клиентов» (создание заявки на обслуживание для Специалистов);
- Специалист - работа с документом «Обслуживание клиентов» (исполнение и закрытие заявки), работа с отчетом «Выработка специалистов»;
- Бухгалтер ИТ фирмы – работа с обработкой «Массовое создание актов» (документ «Реализация товаров и услуг»), работа с отчетом «Анализ выставленных актов»;
- Кадровик-расчётчик – работа с документами «Начисление зарплаты», «Выплата зарплаты», «Начисление фиксированной премии», «График отпусков», работа с регистрами сведений «Графики работ», «Условия оплаты сотрудникам» (ввод информации о заработной плате и плановых отпусках), работа с отчетами «Выработка специалистов», «Расчеты с сотрудниками», «Расход запланированных отпусков».
### Инструкция по заполнению полей обработки «Заполнение графика»:
- поле Период – указать период, на который заполняется график;
- поле Выходные дни – указать порядковые номера выходных дней недели через запятую (для пятидневки: 6,7 – суббота и воскресенье);
- поле «Графики работ» - значение из справочника «Графики работ»;
Обработка «Заполнение графика» предназначена только для учета отработанного времени в днях. Для почасовой оплаты труда данная обработка не применима.
